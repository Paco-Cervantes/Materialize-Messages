Función jquery para mensajes con Materializecss

Introducción

materializeMessages es solo un snippet para la creación de mensajes modales, utilizando el framework Materializecss.

Los mensajes modales ya están diseñados por el framework, y están pensados para mostrar el html de un div oculto, y solo aceptan utilizar una función en el parametro "ready" (Callback for modal open) y en el parametro "complete" (callback for modal close).

Para los dialogos el framework utiliza los Toast. Todo muy apegado a los conceptos de diseño de google y con el estandar mobile first, y si bien esto es fantástico, se echa de menos poder enviar un mensaje al usuario y tener un par de botones para ejecutar alguna acción. sin tener esos divs escondidos para cada mensaje que quieras enviar.

Materializecss (Css)

Utilizando todos los estilos configurables para el mensaje. Las opciones de estilos son las determinadas por Materializecss, para el color primario puedes usar "red, blue, yellow, purple, pink, etc." y el color secundario "darken-2, lighten-3, accent-1, etc." puedes consultar la paleta de colores de Materializecss. Para el tipo de icono (iconType) puedes usar los iconos de google.
<a href="someFooPage.html" class="btn orange" id="example4">example 4</a>

Pasando variables al mensaje, se pueden pasar variables al titulo y al mensaje, utilizemos un ejemplo practico de borrado de usuario, en el enlace que usarias para eliminar el usuario se deben configurar las variables que quieres enviar al mensaje. En este caso en el botón configuraremos lo siguiente: data-id="24", data-firstname="Jhon", data-lastname="Smith". Estás variables deben ser configuradas en la llamada a la función en javascript con los párametros titleVariables y messageVariables.
<a href="someFooPage.html" class="btn orange" id="example5" data-id="24" data-firstname="Jhon" data-lastname='Smith'>example 5</a>
    /* Button example5 */
						    $('#example5').materializeMessages({
						        dismissible : false,
						        iconType : 'warning',
						        iconColor : 'red',
						        iconSubcolor : 'darken-2',
						        titleVariables : {
						          firstname : 'data-firstname'
						        },
						        title : 'Está seguro de borrar a "{{firstname}}"',
						        titleColor : 'red',
						        titleSubColor : 'darken-2',
						        messageVariables : {
						          firstname : 'data-firstname',
						          lastname : 'data-lastname'
						        },
						        message : 'El usuario {{firstname}} {{lastname}} será borrado permanentemente',
						        messageColor : 'red',
						        messageSubColor : 'darken-4',
						        button1 : true,
						        button1Content : {
						          text : 'Cancelar',
						          action : 'closeMessage'
						        },
						        button2 : true,
						        button2Content : {
						          text : 'Borrar',
						          action : function(elem){ alert ('se borrá al usuario '+elem['data-firstname']+' '+elem['data-lastname'] +' con el id: '+elem['data-id'])}
						        }
						    });
