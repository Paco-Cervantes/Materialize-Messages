Función jquery para mensajes con Materializecss

Introducción

materializeMessages es solo una función para la creación de mensajes modales, utilizando el framework Materializecss.

Los mensajes modales ya están diseñados por el framework, y están pensados para mostrar el html de un div oculto, y solo aceptan utilizar una función en el parametro "ready" (Callback for modal open) y en el parametro "complete" (callback for modal close).

Para los dialogos el framework utiliza los Toast. Todo muy apegado a los conceptos de diseño de google y con el estandar mobile first, y si bien esto es fantástico, se echa de menos poder enviar un mensaje al usuario y tener un par de botones para ejecutar alguna acción. sin tener esos divs escondidos para cada mensaje que quieras enviar.

Pasando variables al mensaje, se pueden pasar variables al titulo y al mensaje, utilizemos un ejemplo practico de borrado de usuario, en el enlace que usarias para eliminar el usuario se deben configurar las variables que quieres enviar al mensaje. En este caso en el botón configuraremos lo siguiente: data-id="24", data-firstname="Jhon", data-lastname="Smith". Estás variables deben ser configuradas en la llamada a la función en javascript con el párametros "variables".

Por ejemplo:
html = <a href="" data-username="Dennis">Enlace</a>
Javascript = variables : { username : 'data-username' }, title : 'aquí se muestra al usuario {{username}}'
Donde "username" será el nombre de la variable que utilizarás en la definición del título o del mensaje y "data-username" el nombre del atributo del cual será extraido el valor de la variable. Para este ejemplo se mostrará en el título "aquí se muestra al usuario Dennis". 

<a href="someFooPage.html" class="btn orange" id="example5" data-id="24" data-firstname="Jhon" data-lastname='Smith'>example 5</a>
/* Button example5 */
    						$('#example5').materializeMessages({
						        dismissible : false,
						        iconType : 'warning',
						        iconColor : 'red',
						        iconSubcolor : 'darken-2',
						        variables : {
						          firstname : 'data-firstname',
						          lastname : 'data-lastname'
						        },
						        title : 'Está seguro de borrar a "{{firstname}}"',
						        titleColor : 'red',
						        titleSubColor : 'darken-2',
						        message : 'El usuario {{firstname}} {{lastname}} será borrado permanentemente',
						        messageColor : 'red',
						        messageSubColor : 'darken-4',
						        button1 : true,
						        button1Content : {
						          text : 'Cancelar',
						          action : 'closeMessage'
						        },
						        button2 : true,
						        button2Content : {
						          text : 'Borrar',
						          action : function(elem){ alert ('aquí debe ir una llamada ajax para borrar al usuario '+elem['data-firstname']+' '+elem['data-lastname'] +' con el id: '+elem['data-id']);}
						        }
						    });

Configuraciones

Todas las opciones de configuración recibidas por la función.

Parámetro Descripción Tipo Default Ejemplo
iconType Define el tipo de ícono a presentar antes del título. Solo se aceptan referencia a los iconos de google. string info error, account_box, get_app, home, etc.
iconColor Define el color del ícono del título, solo acepta colores definidos en la paleta de colores de Materializecss. string grey purple, yellow, deep-orange, blue, etc.
iconSubColor Define la clase de color opcional del ícono del título, solo acepta las clases opcionales definidos en la paleta de colores de Materializecss. string darken-2 accent-4, lighten-2, darken-3, etc.
title Texto a mostrar como título del mensaje. string | html dont forget set the title <span>Cualquier Texto de título</span>
titleColor Color del texto a mostrar como título del mensaje. string grey purple, yellow, deep-orange, blue, etc.
titleSubColor Color opcional del texto a mostrar como título del mensaje. string darken-2 accent-4, lighten-2, darken-3, etc.
message Texto del mensaje a mostrar. string | html dont forget set the message <span>Cualquier texto para el mensaje.</span>
messageColor Color del texto del mensaje a mostrar. string grey purple, yellow, deep-orange, blue, etc.
messageSubColor Color opcional del texto del mensaje a mostrar. string darken-3 accent-4, lighten-2, darken-3, etc.
bgMessage Color de fondo del contenedor del título y el mensaje. string white purple, yellow, deep-orange, blue, etc.
bgMessageSubcolor Color opcional de fondo del contenedor del título y el mensaje. string ' ' accent-4, lighten-2, darken-3, etc.
bgButtonsContainer Color de fondo del contenedor de los botonos del mensaje. string white purple, yellow, deep-orange, blue, etc.
bgButtonsContainerSubcolor Color opcional de fondo del contenedor de los botonos del mensaje. string ' ' accent-4, lighten-2, darken-3, etc.
button1 Indica si el botón 1 será utilizado en el mensaje. boolean false true | false
button2 Indica si el botón 2 será utilizado en el mensaje. boolean false true | false
dismissible Indica si el mensaje podrá ser cerrado al hacer click fuera del mismo. boolean true true | false
variables Recibe las variables que serán buscadas y reemplazadas en el título o contenido del mensaje. object empty object firstname : 'data-firstname'
autoclose Indica si el mensaje deberá cerrarse automáticamente. boolean false true | false
secondsToClose Número de segundos en que se cerrará automáticamente el mensaje. number 3 5
modalType Indica el tipo de modal en el que se mostrará el mensaje. string message message | bottom
